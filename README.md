# Proyecto QA Automation - SauceDemo

##Propósito del proyecto:

Este proyecto tiene como objetivo automatizar pruebas funcionales sobre el sitio SauceDemo utilizando Python, Selenium WebDriver y Pytest.

Las automatizaciones desarrolladas permiten validar funcionalidades principales de la aplicación, como:

- Login de usuario
- Navegación del inventario
- Visualización de productos
- Agregado de productos al carrito
- Verificación de elementos de interfaz

---

##Tecnologías utilizadas:

- Python
- Selenium WebDriver
- Pytest
- Pytest-HTML
- Visual Studio Code
- Git y GitHub
